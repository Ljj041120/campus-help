========================================
一、项目里每个文件夹是干什么的
========================================

backend/          → 后端代码（Java Spring Boot，不用打开看）
admin/            → 管理后台（Vue 3，不用打开看）
uniapp-vue3/      → 微信小程序（UniApp，用 HBuilderX 打开）
database/         → 数据库建表脚本（已经执行过了，不用管）
README.md         → 项目部署说明
使用说明.md       → 本文件

========================================
二、用什么软件打开什么代码
========================================

看后端代码（Java）：
  打开 VS Code → 文件 → 打开文件夹 → 选中 backend

看管理后台代码（Vue）：
  打开 VS Code → 文件 → 打开文件夹 → 选中 admin

看小程序代码：
  打开 HBuilderX → 文件 → 打开目录 → 选中 uniapp-vue3/uniapp

注意：VS Code 和 HBuilderX 只是用来看代码和改代码的，
不是用来运行的。运行要按照下面的步骤。

========================================
三、每次开机后的启动步骤（按顺序来）
========================================

第1步：MySQL（一般不用管，已经自动启动了）

如果没启动：
  按键盘左下角 Win 键
  输入 services.msc 回车
  找到 MySQL84
  右键 → 启动

第2步：启动 Redis

  按 Win + R
  弹出窗口里输入 cmd
  回车
  桌面上出现一个黑色窗口
  在黑色窗口里粘贴下面这行字，按回车：
  "C:\Users\ASUS\Redis\redis-server.exe"
  看到一个图标，这个窗口不要关

第3步：启动后端

  按 Win + R
  输入 cmd
  回车（又出现一个黑色窗口，和刚才那个不一样）
  在黑色窗口里粘贴下面这行，按回车：
  cd C:\Users\ASUS\Desktop\综合实训1
  再粘贴下面这行，按回车：
  java -jar backend\target\campus-help-backend-1.0.0.jar
  看到 Tomcat started on port 8080 就成功了
  这个窗口不要关

第4步：启动管理后台

  按 Win + R
  输入 cmd
  回车（再开一个黑色窗口，现在是第3个了）
  粘贴下面这行，按回车：
  cd C:\Users\ASUS\Desktop\综合实训1
  再粘贴下面这行，按回车：
  npm --prefix admin run dev
  看到 http://localhost:5173 就成功了
  这个窗口不要关

第5步：启动微信小程序

  双击桌面上的 HBuilderX 图标
  点顶部菜单「文件」
  点「打开目录」
  选中桌面 → 综合实训1 → uniapp-vue3 → uniapp 这个文件夹
  点顶部菜单「运行」
  点「运行到小程序模拟器」
  点「微信开发者工具」
  等几秒，微信开发者工具会自动打开
  在小程序里点「测试登录（免微信授权）」

完成！你现在应该能看到小程序首页了。

  账号：admin

========================================
五、关机前要关掉的
========================================

1. 关掉微信开发者工具（直接点X）
2. 关掉 HBuilderX（直接点X）
3. 第1个黑色窗口（Redis）→ 按 Ctrl+C
4. 第2个黑色窗口（后端）→ 按 Ctrl+C
5. 第3个黑色窗口（管理后台）→ 按 Ctrl+C
6. 关电脑

========================================
六、常见问题及解决方法
========================================

问题1：浏览器打开 http://localhost:5173 显示不了

  原因：管理后台没启动
  解决：检查有没有第4步那个黑色窗口
       如果没有，重新执行第4步

问题2：管理后台打开后数据都是0

  原因：后端没启动
  解决：检查有没有第3步那个黑色窗口
       如果没有，重新执行第3步
       如果有但显示报错，截图给我看

问题3：微信开发者工具里点登录没反应

  原因：后端没启动，连不上
  解决：检查第3步那个黑色窗口还在不在
       如果在，看有没有报错

问题4：点「测试登录」提示失败

  原因：后端连不上，或者后端地址不对
  解决：打开 uniapp-vue3/uniapp/api/config.js
       看第一行 baseUrl 是不是 http://10.100.153.113:8080/api
       如果不是，改成这个

问题5：微信开发者工具报错 Cannot set property 'query'

  原因：微信开发者工具自己的bug，不是你的代码问题
  解决：不用管，不影响使用

问题6：手机扫码预览打不开

  原因：手机和电脑不在同一个WiFi
  解决：让手机连你电脑连的那个WiFi

问题7：改了代码后怎么看到效果

  后端代码：
    第3步那个窗口按 Ctrl+C
    重新执行第3步的两条命令

  管理后台代码：
    第4步那个窗口会自动刷新
    如果没有，按 Ctrl+C 重新执行第4步

  小程序代码：
    在 HBuilderX 按 Ctrl+S 保存
    等几秒自动编译，微信开发者工具自动刷新

问题8：MySQL 连不上

  解决：按 Win 键 → 输入 services.msc → 回车
       找到 MySQL84 → 右键 → 启动

问题9：Redis 连不上

  解决：看看第2步那个黑色窗口还在不在
       如果在但显示报错，关掉重新执行第2步

问题10：全部关掉后下次怎么打开

  按「三、启动步骤」从第1步到第5步再来一遍

| 项目信息 | 内容 |
|---------|------|
| **产品名称** | 校园帮 - 互助跑腿平台 |

## 🏗️ 技术架构

```
用户端 (UniApp)         管理后台 (Vue 3 + ECharts)
       │                       │
       └──────────┬────────────┘
                  │ HTTP/HTTPS
          ┌───────▼────────┐
          │   Spring Boot  │
          │   + Security   │
          └───┬───────┬────┘
              │       │
      ┌───────▼──┐ ┌──▼────────┐
      │  MySQL   │ │  Redis    │
      │  业务数据 │ │ GEO/锁/缓存│
      └──────────┘ └───────────┘
```

| 层级 | 技术选型 |
|------|---------|
| 后端框架 | Spring Boot 3.2.5 + MyBatis-Plus 3.5.6 |
| 安全认证 | Spring Security + JWT + BCrypt |
| 数据库 | MySQL 8.0 + Redis 6.0+ (GEO/锁/缓存) |
| 用户端 | UniApp (Vue 3) — 微信小程序 |
| 管理后台 | Vue 3 + Element Plus + ECharts 5 |
| 支付 | 支付宝沙箱 (alipay-sdk-java 4.15) |
| 工具库 | Hutool 5.8.25 + Lombok |
